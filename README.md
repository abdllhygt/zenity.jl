# zenity.jl
Cross-platform dialog boxes for Julia language

![alt info](https://image.ibb.co/h9qjQT/0j.png)

*(Elementary OS)*
```julia
Zenity.wsize(300,200);Zenity.info("Programming GUI with <b>Julia</b>")
```

![alt info](https://image.ibb.co/nxzeLT/2.png)
